# BrowserStack Cross-Browser Testing

## Overview

This project focuses on cross-browser testing of the OrangeHRM web application using Selenium WebDriver, Python, and BrowserStack.

## Objective

The objective of this project is to verify that the application's login functionality works consistently across different web browsers.

## Browsers Tested

* Google Chrome
* Mozilla Firefox
* Microsoft Edge

## Activities Performed

* Automated login testing using Selenium WebDriver.
* Executed test cases on multiple browsers using BrowserStack.
* Verified browser compatibility and application behavior.
* Validated successful login functionality across browsers.

## Tools Used

* Selenium WebDriver
* Python
* BrowserStack

* Functional Testing

Successfully executed automated test cases across multiple browsers and verified consistent application behavior and functionality.
